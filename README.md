# token-frog

A Figma plugin for design token management.

> Not yet approved on the Figma Community? No problem — you can install it directly from this repo in under a minute.

---

## How to Install (Manual / Developer Mode)

### Step 1 — Download the plugin files

Click the green **Code** button on this page → **Download ZIP**, then unzip it somewhere on your computer (e.g. your Desktop).

### Step 2 — Open Figma Desktop

Manual plugin installation only works in the **Figma desktop app** (not the browser). Download it at [figma.com/downloads](https://www.figma.com/downloads) if you haven't already.

### Step 3 — Load the plugin

1. Open any Figma file
2. Go to **Menu** (the Figma logo, top-left) → **Plugins** → **Development** → **Import plugin from manifest…**
3. Navigate to the folder you unzipped and select the **`manifest.json`** file
4. Click **Open**

### Step 4 — Run it

Go to **Menu → Plugins → Development → token-frog** to launch the plugin.

That's it — the plugin will now appear in your Development plugins every time you open Figma.

## Files

| File | What it does |
|------|-------------|
| `manifest.json` | Tells Figma about the plugin |
